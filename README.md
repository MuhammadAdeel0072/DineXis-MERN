# AK-7 Rest - A Premium MERN Stack Dining Experience.

## What is AK-7 Rest?
AK-7 Rest is a modern web application built for the food industry. Think of it as a digital bridge between a hungry customer and a gourmet kitchen. 

We designed it as a full-stack restaurant application allowing users to browse categories (Drinks, Food, Dishes, Sweets) and order online.

## How it works
- **Welcome:** The user arrives at a luxury-themed homepage.
- **Identity:** They log in easily using Google or Twitter (no need to remember a new password).
- **Discovery:** They browse high-quality images of food and drinks.
- **Action:** They add items to a cart and place an order.

## HCI Features
Highlighting the **Human-Computer Interaction (HCI)** focus:
- **Responsive design**: Flawless experience across devices.
- **Dark/Gold luxury theme**: "Midnight Gourmet" theme reduces eye strain while making the food colors pop and establishing a premium "vibe". Gold accents direct attention toward primary calls to action.
- **Interactive Feedback**: Features glassmorphism depth, frosted glass blur, and glow effects on hover.
- **Frictionless Social Authentication**: Effortless sign in process without cognitive load.

## The Technical Breakdown

- **Frontend (The Look):** Built with React and Vite 8. We used Tailwind CSS to create our custom "Midnight Gourmet" visual identity.
- **Authentication (The Guard):** Powered by Clerk. This handles the complicated math of security so users can log in via Email OTP or Social OAuth natively. 
- **Backend (The Brain):** A Node.js and Express server that talks to the database.
- **Database (The Memory):** MongoDB stores all the food items, user profiles, and order history.

## Key Features
- Social Login (Google/X)
- Email OTP
- Shopping Cart
- Category-based filtering
